PokeBattler
---------------------------------------------

Use arrow keys to select moves and pokemon

Press enter to select the choice

M to open move pannel
S to open switch choices 

Run moves.py

Install curses, sklearn, and pandas

moves.py shows list of possible moves and pokemon.py shows pokemon
